# FarEye ANZ GTM Command Center

A premium static website prototype for FarEye ANZ market expansion intelligence.

## How to preview
Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Structure
- Multi-page static HTML site
- Shared CSS and JavaScript
- Animated hero, counters and cards
- Interactive ANZ opportunity map
- Searchable and filterable target account database
- Segment-specific dashboards
- CRO strategy page
- Resource hub with clean document views

## Production notes
Connect forms and download buttons to Wix CMS, CRM, or document storage for production deployment.
